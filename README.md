### Unity Version  
2023.2.20f1  

### Controls  
- **`Mouse`**: Camera movement  
- **⬅️ or `A`**: Move left  
- **➡️ or `D`**: Move right  
- **⬆️ or `W`**: Move forward  
- **⬇️ or `S`**: Move backward  
- **`Space`**: Use Lidar Scanner  
- **`Q`**: Place Crystals  
- **`E`**: Interact  
- **`LMB`**: Punch  
- **`Esc`**: Pause Screen
